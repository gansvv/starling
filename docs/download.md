
## Download binaries ##

Download the latest binaries from the [releases](https://github.com/iot-for-all/starling/releases).

Download the zip file specific to your platform and unzip into a folder:
1. Windows (AMD64)
2. MacOS (AMD64)
3. Linux (AMD64 or ARM64)
